# Посібник користувача системи PetHaven

Цей документ описує правила взаємодії з консольним інтерфейсом адміністратора готелю для тварин.

##  Підготовка та запуск програми
При старті застосунку інфраструктурний шар автоматично зчитує файли `rooms.json` та `bookings.json` із робочої директорії програми. Якщо файли відсутні, система створить їх автоматично з порожніми колекціями.

##  Доступні операції в меню

### 1. Опція "Заселення тварини" (Check-In)
Використовується для реєстрації нового постояльця.
* **Вхідні дані:** Ім'я тварини, тип (введіть `0` для котів, `1` для собак), вік, унікальний номер вольєра та тривалість проживання в днях.
* **Логіка роботи:** Система перевірить, чи вільна кімната, і викличе обрану `IPricingStrategy`. На екран буде виведено підсумкову ціну, а кімната зафіксується як зайнята.

### 2. Опція "Виселення тварини" (Check-Out)
Використовується після завершення терміну проживання.
* **Вхідні дані:** Скопіюйте та введіть текстовий унікальний ідентифікатор (`Guid`) бронювання.
* **Логіка роботи:** Статус бронювання перейде в `Completed`, а вольєр знову стане вільним для інших тварин.

### 3. Опція "LINQ Звіти та Аналітика"
Генерує оперативні звіти для адміністратора в реальному часі:
* Поточний загальний фінансовий дохід від усіх проживань.
* Список тварин, які зараз перебувають у готелі.
* Статистика завантаженості готелю у відсотках.
