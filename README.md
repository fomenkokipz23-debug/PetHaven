# PetHaven - Керування готелем для тварин (v1.0.0)

Проєкт є випускним капстоун-релізом курсу ООП. Реалізує асинхронне керування готелем для тварин на базі Чистої Архітектури із впровадженням SOLID принципі та автоматизованого тестування.

##  Навігація по документації
*  [Інструкція користувача (USER_GUIDE.md)](USER_GUIDE.md) - як запускати застосунок та користуватися консольним меню.
*  [Посібник розробника (DEVELOPER_GUIDE.md)](DEVELOPER_GUIDE.md) - архітектурний опис шарів, SOLID, патерни та правила розширення коду.
*  [Звіт про тестування (TESTING.md)](TESTING.md) - інструкція із запуску тестів, Moq-тести та метрики Code Coverage.
*  [Журнал змін (CHANGELOG.md)](CHANGELOG.md) - історія релізів, доданих фіч та виправлених багів.
