# Журнал змін (Changelog) - PetHaven


### Added (Що додано в межах Lab 36-37)
* Впроваджено ієрархію власних доменних винятків `BusinessRuleException` для безпечного захисту інваріантів системи.
* Реалізовано інтеграційні тести життєвого циклу файлового сховища у ізольованих тимчасових папках ОС (`Path.GetTempFileName`).
* Додано інтерактивні XML-коментарі для розробників над публічним API класу `BookingService`.

### Changed (Що змінено/оптимізовано)
* Сервіс `BookingService` повністю переведено на роботу через доменні інтерфейси замість конкретних файлових реалізацій.
* Оптимізовано LINQ-алгоритми аналітики: підрахунок тварин за типами переведено на структуру `Dictionary<PetType, int>` для константного пошуку $O(1)$.

### Fixed (Що виправлено)
* Виправлено критичний збій невідповідності очікуваних винятків (Exception Mismatch) під час спроби скасування вже завершених бронювань у тестах.
* Проведено очищення Git-індексу: з історії відстеження повністю вилучено згенеровані бінарні файли з тимчасових папок компіляції `bin` та `obj`.
