# План релізу v1.0.0 - PetHavenCapston

## 1. Scope Релізу v1.0.0
* **Функціонал:** Повне консольне меню, асинхронне JSON-збереження, 3 Use Cases (Check-In, Check-Out, Cancel), патерн розширення Strategy для розрахунку вартості, LINQ-аналітика фінансів та завантаженості.
* **Якість:** Покриття коду тестами (Unit + Integration) > 85%, ізоляція шарів через `Moq`, власні доменні винятки `BusinessRuleException`.

## 2. Допустимий технічний борг та плани "Після курсу"
* **UI-шар:** Консольне меню залишається не покритим тестами (перенесення UI на WPF/Blazor планується в наступних версіях).
* **Конкурентність:** Файлове сховище не підтримує блокування файлів при одночасній роботі кількох потоків (Race Conditions). Допустимо для локального однокористувацького застосунку.
