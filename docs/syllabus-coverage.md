# Матриця покриття тем курсу ООП - PetHaven

| Група тем курсу | Що і де конкретно реалізовано в коді проєкту | Статус |
| :--- | :--- | :--- |
| **1. Основи ООП** | Написано класи `Pet`, `Room`, `Booking`, інкапсульовано стан через `private set` та автопроперті. | Використано |
| **2. Абстракції та Поліморфізм** | Впроваджено інтерфейси репозиторіїв та поліморфний розрахунок вартості тварин через стратегії. | Використано |
| **3. Generics та Колекції** | Створено універсальний потокобезпечний інфраструктурний клас `JsonFileStore<T>`, де `T : class`. | Використано |
| **4. LINQ та Делегати** | В `AnalyticsService` через лямбда-вирази використано методи `Sum`, `Where`, `GroupBy`, `ToDictionary`. | Використано |
| **5. Асинхронність** | Всі I/O операції з диском переведені на `async/await` (`Task`, `WriteAllTextAsync`, `SaveChangesAsync`). | Використано |
| **6. Робота з помилками** | Створено ієрархію `PetHavenDomainException` та `BusinessRuleException`. | Використано |
| **7. Принципи SOLID** | Повний розподіл шарів. Залежності сервісів розгорнуті строго через інтерфейси домену (DIP). | Використано |
| **8. Патерни проєктування** | Реалізовано архітектурний патерн поведінки **Strategy** (`IPricingStrategy`). | Використано |
| **9. Автоматизоване тестування** | Написано 21 юніт-тест (xUnit, `Theory`, `Moq`) та 8 інтеграційних тестів для файлового шару. | Використано |
